### BrowserOptions
- `headless` <type>{ boolean }</type> headless mode of browser.
- `viewport` <type>{ object }</type> viewport of browser.
  - `width` <type>{ number }</type> width of viewport.
  - `height` <type>{ number }</type> height of viewport.
