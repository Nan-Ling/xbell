## mouse.click

**Signature: mouse.click(x, y, options?)**

- `x` <type>Horizontal position of the mouse.</type>
- `y` <type>Vertical position of the mouse.</type>
- `options?` <type>object</type>
  - `button?` &lt;"left" | "right" | "middle"&gt; Defaults to left.
  - `clickCount?` <type>number</type> defaults to 1. See UIEvent.detail.
  - `delay?` <type>number</type> Time to wait between mousedown and mouseup in milliseconds. Defaults to 0

**Returns** <type>Promise&lt;void&gt;</type>

Shortcut for mouse.move(x, y, options?), mouse.down(options?), mouse.up(options?).

## mouse.down
**Signature: mouse.down(options?)**

- `options?` <type>object</type>
  - `button?` &lt;"left"|"right"|"middle"&gt; Defaults to left.
  - `clickCount?` <type>number</type> defaults to 1. See UIEvent.detail.

## mouse.move

**Signature: mouse.move(x, y, options?)**

- `x` <type>Horizontal position of the mouse.</type>
- `y` <type>Vertical position of the mouse.</type>
- `options?` <type>object</type> Optional object
  - `steps?` <type>number</type>  Defaults to 1. Sends intermediate mousemove events.

**Returns** <type>Promise&lt;void&gt;</type>

Dispatches a `mousemove` event.
